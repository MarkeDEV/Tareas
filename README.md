# 📋 Tareas - Gestor Colaborativo

## 🚀 Funcionalidad
Gestiona tareas de grupo con asignaciones, estados, ranking colaborativo y acceso de administrador protegido.

### 🔧 Características
- **Frontend**: HTML, CSS, JS (ES6).
- **Backend**: Firebase Firestore.
- **Notificaciones**: Soporte para PWA y Service Workers.

---

## ⚙️ Configuración Inicial

### 1️⃣ Requerimientos:
- Node.js >= 18.x
- Firebase Console configurado con credenciales
- `npm` (o `yarn`) instalado

### 2️⃣ Pasos de Inicio:
```bash
# Clonar el repositorio
git clone https://github.com/MarkeDEV/Tareas.git

# Cambiar al directorio
cd Tareas

# Crear archivo .env basado en ejemplo
cp .env.example .env

# Agregar credenciales reales debajo
# Nota: Protege las credenciales privadas siempre
```

### 3️⃣ Uso:
Solo abre `index.html` localmente en tu navegador preferido.

---

## 🚀 Despliegue
1. Hostear en Firebase Hosting:
```bash
firebase deploy
```
2. O usar variables secretas en Cloudflare Pages:
- **Claves:** Admin PIN, Groq AI TOKEN, Firebase Keys.
